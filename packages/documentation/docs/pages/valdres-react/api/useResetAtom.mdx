# useResetAtom

`useResetAtom` is a React hook that takes an atom and returns a function that
will reset the atom back to it's deafult value.

## Example 1 - Reset Count

import { Counter } from "../../../snippets/Counter"

```jsx [Counter.tsx]
// [!code word:useResetAtom]
// [!include ~/snippets/Counter.tsx]
```

:::note[Result]

<div data-title="Preview">
    <Counter />
</div>

:::
