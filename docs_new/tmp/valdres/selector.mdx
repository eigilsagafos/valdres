A `selector` represents derived state from one or more atoms or other selectors.

## Example

```jsx
import { atom, createStore, selector } from "valdres"

const store = createStore()
const temperature = atom(85)
const temperatureInCelcius = selector(get => {
    const f = get(temperature)
    return ((f - 32) * 5) / 9
})

store.get(temperature) // 85
store.get(temperatureInCelcius) // 29.444444444444443
```
