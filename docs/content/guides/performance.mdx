## Why Valdres is faster

The performance advantage comes from a fundamental architectural choice: Valdres uses `useSyncExternalStore` for React integration, while Jotai uses `useEffect` + `useReducer`. This means:

- **No extra renders** — state changes are synchronous, not batched through React's scheduler
- **No effect cleanup overhead** — subscriptions are managed outside React's lifecycle
- **Minimal GC pressure** — WeakMap-based storage with no intermediate objects

Outside of React, the pure JavaScript core is optimized for minimal allocations and cache-friendly data structures.

## Running benchmarks locally

```bash
# Bun (JSC/Safari engine)
bun run --cwd packages/valdres test:bench

# Node.js (V8/Chrome engine)
bun run --cwd packages/valdres test:bench:node
```
