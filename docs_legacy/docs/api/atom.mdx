# atom

An `atom` is a piece of state in Valdres. Atoms can be initialized with default
values, sync or async callbacks or nothing (`undefined`).

## Usage

<Tabs groupId="package">
    <TabItem value="react" label="@valdres/react">
```jsx 
import { atom } from "@valdres/react"

const user = atom({ id: 1, name: "Foo"})

````
    </TabItem>
    <TabItem value="core" label="@valdres/core">

```jsx
import { atom } from "@valdres/core"
````

    </TabItem>

</Tabs>

## Live example

```jsx live noInline
import { atom, useValue, getDefaultStore } from "@valdres/react"
const mousePos = atom([0, 0])
const store = getDefaultStore()

document.addEventListener("mousemove", e => {
    store.set(mousePos, [e.x, e.y])
})

const Comp = () => {
    const [x, y] = useValue(mousePos)
    return (
        <p>
            Your mouse position is x: {x} y: {y}
        </p>
    )
}

render(<Comp />)
```
